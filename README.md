# 🏢 Infraestructura IT 3D

Visualizador 3D interactivo de infraestructura IT con soporte para múltiples pisos, salas y racks.

## ✨ Características

- 🎮 Visualización 3D con Three.js
- 🗄️ Base de datos MySQL con estructura completa
- 🔌 API REST con Node.js/Express
- 📊 Monitoreo de dispositivos (Servidores, Switches, Patch Panels)
- 🔗 Visualización de cables de red
- 📱 Responsive para escritorio y móvil
- 🎨 Interfaz moderna con efectos 3D

## 🚀 Instalación

### Requisitos

- Node.js 14+
- MySQL 5.7+
- Git

### Pasos

1. **Clonar repositorio**

```bash
git clone https://github.com/CERU246/infrastructure-visualizer.git
cd infrastructure-visualizer
```
